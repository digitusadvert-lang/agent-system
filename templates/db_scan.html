Database Duplicate Scan

Slowly crawls every record in the database and groups exact duplicates by IC, phone number, and name.

Start a New Scan

The scan runs in the background and processes records in batches of 500. It will not lock the database. Large databases may take a few minutes.

  • IC duplicates — same IC number (digits only, dashes ignored)
  • Phone duplicates — same phone number (digits only, country code normalised)
  • Name duplicates — exact name match (case-insensitive), only where not already caught by IC/phone
How to Use
  1. Click Start Scan Now — a progress page will appear.
  2. Wait for the scan to finish (progress bar reaches 100%).
  3. You'll be taken to the Report showing all duplicate groups.
  4. For each group, choose an action:
    • Keep First — keep oldest record as master and merge the rest into it
    • Keep Last — keep newest record as master and merge the rest into it
    • Keep This — pick a specific master record and merge the rest into it
    • Merge — choose the best master automatically and preserve unique fields
    • Skip — leave untouched for now
