Setup Guide
  1. Open Telegram and message @BotFather
  2. Type /newbot and follow the steps to create a bot
  3. Copy the Bot Token and paste it above
  4. Start a chat with your new bot (send any message)
  5. Visit api.telegram.org/bot<TOKEN>/getUpdates to get your Chat ID
  6. Set your Telegram webhook URL to:
    https://yourdomain.com/telegram/webhook
    (For localhost testing use ngrok)
  7. Enable the toggle and save
Local testing: Run ngrok http 5000 then register the webhook using the ngrok URL.
